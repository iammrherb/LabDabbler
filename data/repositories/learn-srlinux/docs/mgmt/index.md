# SR Linux Management

* Open-sourced, fully customizable and feature-rich [CLI](../cli/index.md).
* High-performant, telemetry-optimized gNMI.
* Friendly and easy-going [JSON-RPC](../blog/posts/2022/json-rpc-basics.md) interface.
